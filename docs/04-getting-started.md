# Getting Started

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed (version >= 14).
- You may need to install [npm](https://www.npmjs.com/) to manage packages.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Jannescynatix/schul-dashboard.git
   cd schul-dashboard
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## Environment Setup
- Create a `.env` file in the root of your project and set the necessary environment variables. You can refer to `.env.example` for the required variables.

## Running the Development Environment
- Start the development server:
   ```bash
   npm run dev
   ```
- Open your browser and go to `http://localhost:3000` to see the application in action.
